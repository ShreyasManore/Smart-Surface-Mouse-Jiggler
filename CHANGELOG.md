# Changelog

All notable changes to this project are documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-07-02

### Added
- Initial public release of the Smart Surface Mouse Jiggler hardware design
- Complete circuit and block diagrams
- Full bill of materials and components list
- Assembly guide and wiring guide
- Technical documentation covering power consumption, battery runtime, and performance
- Testing report with methodology and results
- Repository documentation: README, CONTRIBUTING, CODE_OF_CONDUCT, PROJECT_STRUCTURE
- MIT License

### Notes
- This is the first stable hardware revision. Enclosure is a basic protective housing; a 3D-printed version is planned for a future release (see [Future Scope](README.md#future-scope)).
