# Specifications

## Electrical Specifications

| Parameter | Value |
|---|---|
| Operating Voltage | 3.0–4.2 V DC (18650 single-cell range) |
| Nominal Voltage | 3.7 V |
| Current Draw (steady-state) | 95–150 mA |
| Peak/Stall Current | Up to ~300 mA (motor-dependent) |
| Battery Capacity (typical) | 2200–3500 mAh |
| Estimated Runtime | 15–30 hours (capacity-dependent) |

## Mechanical Specifications

| Parameter | Value |
|---|---|
| Enclosure Dimensions (baseline) | ~90 mm x 90 mm x 30 mm |
| Disc Diameter | ~70 mm |
| Disc Rotation Speed | 10–30 RPM |
| Total Assembled Weight | ~120–180 g (component dependent) |
| Switch Type | SPST toggle/slide |

## Environmental Specifications

| Parameter | Value |
|---|---|
| Recommended Operating Temperature | 10°C – 35°C |
| Recommended Humidity Range | 20% – 70% non-condensing |
| Intended Use Environment | Indoor, desk/office setting |

## Compatibility

| Item | Compatibility Notes |
|---|---|
| Mouse Type | Optical or laser sensor mice (wired or wireless) |
| Mouse DPI Range Tested | 800–3200 DPI |
| Operating Systems | OS-agnostic (interacts only via standard mouse HID, not software) |
