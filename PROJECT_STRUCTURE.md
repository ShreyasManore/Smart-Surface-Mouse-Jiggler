## Directory Purpose Summary

| Directory | Purpose |
|---|---|
| `images/` | All static diagrams and photos referenced throughout the documentation |
| `docs/` | Narrative and reference documentation (reports, guides, testing) |
| `hardware/` | Hardware-specific technical references (circuits, BOM, specs) |
| `media/` | Video and animated demo assets |
