# Contributing to Smart Surface Mouse Jiggler

Thank you for your interest in improving this project. Contributions of all kinds are welcome — hardware revisions, documentation improvements, testing data, and enclosure designs.

## Ways to Contribute

- **Hardware improvements**: motor selection, disc texture optimization, enclosure redesign
- **Documentation**: clarifying assembly steps, fixing errors, adding diagrams
- **Testing**: reporting compatibility results across different mouse models and operating systems
- **Future Scope features**: PWM speed control, charging module integration, automatic detection logic

## Getting Started

1. Fork the repository.
2. Create a feature branch:
   ```bash
   git checkout -b feature/your-improvement
   ```
3. Make your changes, keeping documentation and diagrams in sync with any hardware modifications.
4. Commit with a clear, descriptive message:
   ```bash
   git commit -m "Add: adjustable disc mount for variable motor sizes"
   ```
5. Push your branch and open a Pull Request against `main`.

## Pull Request Guidelines

- Clearly describe the motivation and the change.
- If modifying hardware specs, update the relevant files under `hardware/` and `docs/`.
- If adding new images or diagrams, place them in `images/` and reference them with relative paths.
- Keep documentation professional, concise, and consistent with the existing tone.
- One logical change per pull request where possible.

## Reporting Issues

When filing an issue, please include:
- A clear description of the problem or suggestion
- Steps to reproduce (for hardware/behavioral issues)
- Photos or diagrams if relevant
- Your test environment (OS, mouse model, motor/battery specs used)

## Code of Conduct

By participating in this project, you agree to abide by the [Code of Conduct](CODE_OF_CONDUCT.md).

## Questions

Open a GitHub Issue with the `question` label, and it will be addressed as soon as possible.
